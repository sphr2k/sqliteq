# AGENTS

This repository uses **bun** for local development workflows.

- Use `bun install` for dependencies.
- Treat `bun.lock` as the canonical lockfile.
- Use `bun run build` to build.
- Use `bun run test` to run the test suite.
- Use `bun run bench` for benchmarks.
- Do not use `npm`, `yarn`, or `pnpm` for routine repo workflows.
